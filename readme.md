## Todo

A simple Ruby todo list.

- Can add, list and delete contents
- Application run through IRB
- Clone repo and run 'bundle install'

Then type:
- IRB
- require './lib/todo_list'
- todo = Todo.new

To add an item:

- todo.add("item")

To delete an item:

- todo.delete("item")

To display all items:

- todo.list

To clear the list:

- todo.clear
